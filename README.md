# 悬浮球交付物验收工具

本工具是一个本地 HTML 页面，用于审核悬浮球物料交付文件夹或 ZIP 压缩包。

## 使用方式

1. 打开 `index.html`，或访问本地服务地址。
2. 点击「选择交付文件夹」或「上传 ZIP 压缩包」。
3. 选择 `XX活动悬浮球物料交付日期@负责人` 这一层文件夹，或包含这一层文件夹的 ZIP。
4. 查看自动检查结果，并完成右侧人工确认项。
5. 复制或下载验收报告。

ZIP 会在本地浏览器内解析，不上传文件。当前支持常见的无压缩和 Deflate 压缩 ZIP。

## 已覆盖规则

- 只交付 1 个文件夹。
- 文件夹命名包含「悬浮球物料交付」、日期和负责人。
- 文件夹内包含常规态和吸边态。
- 常规态与吸边态按名称成对交付。
- 文件名中识别明显无意义命名。
- 常规态 JSON 不超过 100KB，超过 40KB 给出压缩建议。
- JSON 可正常解析。
- 吸边态 PNG 尺寸为 99 x 300。

## 需要人工确认

- 常规态首尾帧是否保持一致。
- 常规态首帧正面利用率是否最大化。
- 是否已在运营悬浮球内审群完成确认。
