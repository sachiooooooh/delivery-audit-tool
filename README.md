# 悬浮球交付物验收工具

本工具是一个本地 HTML 页面，用于审核悬浮球物料交付文件夹或 ZIP 压缩包。

## 使用方式

1. 打开 `index.html`，或访问本地服务地址。
2. 点击「选择交付文件夹」或「上传 ZIP 压缩包」，也可以把文件夹或 ZIP 直接拖到对应入口。
3. 选择 `XX活动悬浮球物料交付日期@负责人` 这一层文件夹，或包含这一层文件夹的 ZIP。
4. 查看自动检查结果，并完成右侧人工确认项。
5. 复制或下载验收报告。

ZIP 会在本地浏览器内解析，不上传文件。当前支持常见的无压缩和 Deflate 压缩 ZIP。

人工确认区会直接内嵌常规态预览，用于检查首尾帧衔接和首帧展示效果。若交付物没有 `demo/data.html`，工具会尝试用 `data.json` 生成预览；此时需要浏览器能访问 Lottie 播放器 CDN。

常规态预览使用固定 `180 x 180` 动效外框和居中的 `150 x 150` 首帧安全框。默认停在首帧，点击预览后播放 1 次，播完停在末帧，方便人工判断首帧利用率、动效是否越界和首尾帧衔接。

## 已覆盖规则

- 只交付 1 个文件夹。
- 文件夹命名包含「悬浮球物料交付」、日期和负责人。
- 文件夹内包含常规态和吸边态。
- 常规态与吸边态按名称成对交付。
- 文件名中识别明显无意义命名。
- 常规态 JSON 不超过 100KB，超过 40KB 给出压缩建议。
- JSON 可正常解析。
- 吸边态 PNG 尺寸为 99 x 300。

## 需要人工确认

- 常规态首尾帧是否保持一致。
- 常规态首帧正面利用率是否最大化。
